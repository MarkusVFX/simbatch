
1 go to https://www.python.org/downloads/release/python-2713/


2 download Windows X86-64 MSI Installer  (https://www.python.org/ftp/python/2.7.13/python-2.7.13.amd64.msi)


3 run installer: python-2.7.13.amd64.msi


4 Install PySide with PIP*
	c:\Python27\Scripts\pip install -U PySide

	
5a install SimBATCH with PIP*
  c:\Python27\Scripts\pip install -U PySide
  
5b install SimBATCH  manual installation
	go to https://github.com/MarkusVFX/simbatch
	find clone or download button
	download zip
	
	or
	
	download directly: https://codeload.github.com/MarkusVFX/simbatch/zip/master
	
	
	unpack directory  simbatch  to   your_dir	
	
	run command:  your_dir\SimBatch.py
